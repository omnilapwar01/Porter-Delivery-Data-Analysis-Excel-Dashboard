 📦 Porter Data Analysis Dashboard

An end-to-end Excel analytics project focused on analyzing Porter business data to generate meaningful insights through dashboards, Pivot Tables, charts, and Excel functions.

 📌 Project Overview

This project demonstrates how Microsoft Excel can be used to analyze business data and convert raw datasets into meaningful insights. The analysis focuses on identifying trends, summarizing key metrics, and building an interactive dashboard for better decision-making.
